== Description ==

TID Custom Login Page plugin can manage to customize your WordPress Login page. You may update Image and color. Also You can change background Image and background overlay from back end. 

Plugin documentation: [https://wordpress.org/plugins/tid-custom-login-page/](https://wordpress.org/plugins/tid-custom-login-page/)

== You may changes the settings ==
* Change WordPress Default Logo.
* Change Primary Colors.
* Change Secondary Colors.
* Change Background Image .

= Usage =

* Go to your Dashboard after installation and navigate to "Login Option" to configure the accordion.
* Change the Primary Color for the Login Button Color and Input Border Color.
* Change the Secondary Color for the Go To Back Button
* Upload Logo and Background Image. You can not directly upload images here. First, upload the image in Media and copy the media link and paste the media link here. 
* You can Change the Background Image Overlay Color. Just input 0.1 to 0.9 in the Background brightness area.

= Features =

  * Very easy installation.
  * Unlimited Colors option.
  * Custom login page logo option.
  * Custom logo image.
  * Lightweight and fast.
  * Flexible and easy to use.

== Installation ==
1. Upload 'tid-custom-login-page' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
